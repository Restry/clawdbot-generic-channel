使用指南

快速开始

  1. 点击右上角 ⚙️ 设置 按钮
  2. 最省事的方式是直接粘贴完整连接 URL，或扫码导入
  3. 如果手动填写，再输入 WebSocket 服务器地址（例如：ws://localhost:8080/ws）
  4. 填写用户名；如果服务端启用了认证，再填写连接 Token；初始会话 ID 可选
  5. 点击 连接 按钮
  6. 连接成功后，在顶部会话栏里打开一个现有会话，或直接新建一个会话再开始聊天

功能说明

快速导入 支持粘贴完整 WebSocket URL 或 openclaw://connect?... 连接包

扫码登录 支持摄像头扫码和二维码图片导入

多会话 同一用户连接后，可以在顶部会话栏中切换不同 chatId

文本消息 在输入框输入文字，按回车或点击发送按钮

图片消息 点击 🖼️ 按钮选择图片，支持预览后发送

音频文件 点击 🎵 按钮选择本地音频文件，支持预览后发送

语音消息 点击 🎤 按钮开始录音，再次点击停止，支持预览后发送

Agent 选择 连接后可在顶部下拉框切换到不同的服务端 agent

自动重连 连接断开后会自动尝试重新连接

历史记录 自动保存最近的连接配置，方便快速切换

服务器配置

确保您的 OpenClaw 服务已启动并配置了 Generic Channel：

openclaw config set channels.generic-channel.enabled true
openclaw config set channels.generic-channel.connectionMode websocket
openclaw config set channels.generic-channel.wsPort 8080

消息协议

本客户端遵循 Generic Channel 消息协议，支持：

  • 文本消息：messageType: "text"
  • 图片消息：messageType: "image"，通过 Data URL 传输
  • 音频消息：messageType: "audio"，通过 Data URL 传输
  • 语音消息：messageType: "voice"，WebM 格式
  • 思考状态：实时显示 AI 思考过程

常见问题

Q: 连接失败怎么办？

A: 请检查服务器地址是否正确，确保 OpenClaw 服务正在运行，检查防火墙设置。

Q: 支持哪些浏览器？

A: 现代浏览器均支持，推荐使用 Chrome、Firefox、Safari 或 Edge 最新版本。

💡 提示： 首次使用建议阅读完整使用指南。连接配置会自动保存到浏览器本地存储中。
