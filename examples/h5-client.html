使用指南

快速开始

  1. 点击右上角 ⚙️ 设置 按钮
  2. 输入 WebSocket 服务器地址（例如：ws://localhost:8080/ws）
  3. 填写您的聊天 ID 和用户名
  4. 点击 连接 按钮
  5. 连接成功后即可开始聊天

功能说明

文本消息 在输入框输入文字，按回车或点击发送按钮

图片消息 点击 🖼️ 按钮选择图片，支持预览后发送

语音消息 点击 🎤 按钮开始录音，再次点击停止，支持预览后发送

自动重连 连接断开后会自动尝试重新连接

历史记录 自动保存最近的连接配置，方便快速切换

服务器配置

确保您的 OpenClaw 服务已启动并配置了 Generic Channel：

openclaw config set channels.generic.enabled true
openclaw config set channels.generic.connectionMode websocket
openclaw config set channels.generic.wsPort 8080

消息协议

本客户端遵循 Generic Channel 消息协议，支持：

  • 文本消息：messageType: "text"
  • 图片消息：messageType: "image"，通过 Data URL 传输
  • 语音消息：messageType: "voice"，WebM 格式
  • 思考状态：实时显示 AI 思考过程

常见问题

Q: 连接失败怎么办？

A: 请检查服务器地址是否正确，确保 OpenClaw 服务正在运行，检查防火墙设置。

Q: 支持哪些浏览器？

A: 现代浏览器均支持，推荐使用 Chrome、Firefox、Safari 或 Edge 最新版本。
