📚 集成步骤

在现有的 H5 客户端中集成所有功能只需 3 步：

  1. 引入资源文件（JS 和 CSS）
  2. 初始化聊天状态
  3. 在消息处理函数中添加事件处理

示例代码：

<!-- 1. 引入资源 --> <link rel="stylesheet" href="whatsapp-features.css"> <script src="whatsapp-features.js"></script> <script> // 2. 初始化状态 window.chatState = { chatId: 'user-123', userId: 'user-123', userName: '张三' }; // 3. 添加功能 UI window.features.reactions.addReactionUI(messageEl, messageId); window.features.status.addStatusIndicator(messageEl, messageId, true); // ... 其他功能 </script>
